Activity 2

What is depth-first search? Depth-First Search is an algorithm for traversing or searching a tree or graph data structure. It starts at the root node and explores as far as possible along each branch before backtracking.

Advantage: DFS has a memory advantage over Breadth-First Search because it only needs to store the nodes along a single branch of the tree or graph at any given time.

Disadvantage: DFS may not find the shortest path to a node, making it less efficient than BFS in certain situations.

 Week 7 - 13 Arduino and Physical Computing Week 14 Recommender Systems

What we covered

Recommender systems enhance online shopping experiences by predicting and suggesting products that align with individual preferences. Utilizing methods such as collaborative filtering, content-based filtering, and hybrid approaches, these systems analyze user behavior and content characteristics to make personalized recommendations. Collaborative filtering leverages user similarities to suggest items, while content-based filtering focuses on item features and user preferences. Hybrid methods combine these to improve accuracy and overcome limitations like the cold start and sparsity problems. These systems not only aid users in finding relevant products but also boost sales and customer satisfaction for providers.
